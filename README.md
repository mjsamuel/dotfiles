# Matt's Dotfiles

## Installation

### Perquisites
macOS:
```sh
xcode-select --install
```

Debian/Ubuntu:
```sh
apt update
apt install curl git sudo
```

### Bootstrap
Clone the repo and run the bootstrap script:
```sh
git clone https://github.com/mjsamuel/dotfiles.git $HOME/Developer/dotfiles
$HOME/Developer/dotfiles/bootstrap
```
